# 🚀 Short Link（短链接生成系统）

> 基于 **Flask + Redis** 实现的短链接生成与跳转服务，具备高并发计数、限流与自动过期能力。

## ✨ 核心特性

- ✅ **高性能映射**：自研 Base62 短码生成算法
- ✅ **原子计数**：Redis INCR 实现无锁 PV 统计
- ✅ **访问安全**：IP 限流 + URL 合法性校验
- ✅ **资源回收**：TTL 自动过期，防止冷数据堆积
- ✅ **工程规范**：RESTful API + 统一异常处理

## 🛠 技术栈

| 层级 | 技术 |
|---|---|
| Web 框架 | Flask |
| 数据存储 | Redis |
| 算法 | Base62 编码 |
| 并发控制 | Redis 原子操作 |
| 安全 | IP Rate Limit |

## 📐 系统设计原理

### 1️⃣ 短码生成（Base62）
采用 **自增 ID + Base62** 方案，避免 Hash 冲突，保证短码长度最短。
